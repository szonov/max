# max

Неофициальный Go-клиент для работы с MAX через WebSocket.

## Статус проекта

🚧 Проект находится в активной разработке.

На текущем этапе API нестабилен и может изменяться без обратной совместимости.

Меняются:
- структура пакетов;
- публичные интерфейсы;
- названия методов и типов.

Проект опубликован для открытого просмотра и обсуждения, но пока не рекомендуется использовать его в продуктивных приложениях.

Если вы планируете использовать библиотеку — следите за изменениями и проверяйте обновления перед использованием.

## О проекте

Цель проекта — предоставить Go-инструменты для создания собственных приложений, работающих с MAX.

Проект не является официальным клиентом MAX и не связан с владельцами сервиса.
